# Deployment

## Local backend deployment

```bash
cd cwd-backend
npm install
cp .env.example .env
docker compose up -d --build
npm run dev
```

Backend default port is `5050`.

## Production direction (current docs)

The backend docs already define an AWS path using:

- EC2 + Nginx + PM2
- RDS MySQL in private subnet
- Secrets Manager for credentials
- GitHub Actions for deploy automation

Reference: `cwd-backend/docs/aws-deployment-guide.md`.

## Production readiness checklist

1. Health endpoint responds (`/health`).
2. CORS origins set for production frontend domain.
3. Secrets are not committed and are loaded via env or secret manager.
4. Database migrations/schema are applied in target environment.
5. Auth endpoints and role checks validated against real Firebase project.
