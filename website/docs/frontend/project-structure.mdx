# Project Structure

The frontend follows a modular React architecture designed for maintainability: shared UI and layout logic live in `common/`, domain pages live in `pages/`, and API/domain state logic is split into `api/`, `services/`, and `hooks/`.

## Directory Overview

```text
cwd-frontend/
├── public/                     # Static assets served directly
├── src/                        # Main application source
│   ├── api/                    # HTTP client primitives and endpoint constants
│   ├── assets/                 # Icons and front-end static resources
│   ├── common/                 # Shared components/contexts/layouts
│   ├── hooks/                  # Domain-level hooks (donors/donations/users)
│   ├── pages/                  # Route page implementations
│   ├── services/               # API domain service layer
│   └── utils/                  # Shared utility helpers
└── website/                    # Docusaurus docs site
```

## Source (`src/`) File-by-File

### App Entry and Global Styles

- `src/main.jsx`: React app mount entry.
- `src/App.jsx`: Route tree and guard composition.
- `src/index.css`: Global baseline styles.
- `src/App.css`: App-level styles.
- `src/firebase-config.js`: Firebase initialization.

### API Layer

- `src/api/client.js`: Request wrapper, auth headers, error handling.
- `src/api/endpoints.js`: Backend endpoint constants/builders.
- `src/api/transformers.js`: API data shape transformation helpers.

### Assets

- `src/assets/icons/eye.svg`: Visible password icon.
- `src/assets/icons/eyeclosed.svg`: Hidden password icon.
- `src/assets/icons/icons.js`: Central icon exports/mapping.

### Common Shared Layer

- `Badge.jsx`: Status/label badge atom.

- `GoogleButton.jsx`: Google auth action button.

- `RequiredStar.jsx`: Required-field indicator.

- `Text.jsx`: Standardized text wrapper.

- `Form.js`: Generic form wrapper.
- `Input.jsx`: Reusable input field.

- `ActionsMenu.jsx`: Contextual action menu.
- `UserProfile.jsx`: User profile summary component.

#### `src/common/components/navigation/`

- `LogoutModal.jsx`: Logout confirmation modal.

#### `src/common/components/organisms/`

- `DeleteConfirmModal.jsx`: Generic delete confirmation modal.
- `DonationModal.jsx`: Donation create/edit modal.

## Dependencies

Key runtime dependencies used in this frontend:

- `react`, `react-dom`: UI framework/runtime.
- `react-router`, `react-router-dom`: Routing.
- `styled-components`: Component-level styling.
- `firebase`: Authentication and identity integration.
- `recharts`: Dashboard charts.
- `lucide-react`: Icon library.

## Tools Utilized

- `vite`: Dev server and build pipeline.
- `vitest` + Testing Library: Unit/component testing.
- `eslint`: Code quality checks.
- `prettier`: Formatting.
